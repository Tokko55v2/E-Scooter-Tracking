Facts about the Scooter API:

State: Active can be everything, I guess it means bookable or not

lastLocationUpdate: format date; says when was the last location update

lastStateChange: format date; says when was the last state change

batteryLevel: getting back a value of the battery inside the scooter. Like 80%

lat: latitude; type float or double

lng: longitude; type float or double

maxSpeed: says what is the maximum speed of the scooter

zoneId: the zone where the scooter is like VIENNA

licencePlate: is unique number

vin: zoneId and licencePlate together like AC12345
